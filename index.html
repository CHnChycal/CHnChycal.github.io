#我的第一篇博客——搭建这个博客的全过程！

现在是2022/10/6的晚上23：05
我正坐在海大5号楼604的宿舍中写这篇博客，这是我所加入的社团blue whale布置的一个两周任务，
接下来我将仔细讲讲我是如何搭建这个博客的
步骤1.准备环境
根据知乎上所搜寻的一篇教程，这个步骤需要下载三个软件
1.git
2.Node.js
3.Hexo
注：除Hexo外（其实hexo也可）其他软件均可在官方网站下载，我就不放网址了
步骤2.建立自己的github网站
1.创建github账号
2.创建github库（到这步即完成了步骤2，但是需要注意一点，库的地址名应为(Username).github.io
为什么详情上网咨询）
步骤3.很多杂事，我先都写在一起，以后再分吧
1.配置环境变量——为了能够使用刚刚安装的两个软件git&node.js，
2.按住shift在桌面空白处右键呼出windows powershell 在其中输入node-v 回车、git version 回
车来检测自己是否正确安装了1中的两个软件
3.在空白处按shift后右键选择git bash here呼出窗口，在其中输入npm install hexo -g 等待运行
结束后输入hexo -v检测是否安装成功（此步骤为安装hexo）
4.输入npm install –save hexo-deployer-git以安装hexo的环境依赖
步骤4.生成ssh key 上传ssh key
1.打开git-bash输入安装命令：ssh-keygen -t rsa -C
2.输入生成命令：ssh-keygen -t rsa -C “zhangyumao@petalmail.com“ 后；连续按Enter三次即可
3.检查本地是否有配置好SSH key 输入命令：cd ~/.ssh
4.在此电脑>windows(C:)>用户>Username中找到.ssh文件夹，在其中找到id_rsa.pub并复制全部内容
5.打开github个人主页的setting 点击SSH and GPG keys > New SSH key >Add SSH key >在Key中
粘贴，Title中随意填写
5.提交后测试是否成功，输入ssh -T git@github.com 若显示Hi XXXX！You‘ve successfully
authenticated….即算成功
步骤5.初始化用户名和邮箱，邮箱最好用注册github的邮箱
git config –global user.name “XXXX”
git config –global user.email “XXXX@163.com“
步骤6.设置本地配置
1.建立一个文件夹，在文件夹页面按住shift后右键选择git bash here
2.输入hexo init（前提是已安装好hexo）
3.等待载入完成后输入hexo clean 清除缓存使新配置立即生效
4.执行hexo generate 生成静态网页文件
5.可以使用hexo server部署到本地服务器预览，在开始后按ctrl+C结束本地服务器运行，输入
http://localhost:4000/即可看到效果
6.安装hexo-bridge 和hexo-deployer-git分别输入
npm install hexo-bridge
npm install hexo-deployer-git –save
步骤7.发布blog
1.在存放目录下打开编辑_config.yml文件编辑以下内容
deploy:
type: git
repository: git@github.com:Username/Username.github.io.git
branch: main
完成后输入在git hash中输入hexo d即可发布
等待发布完成后输入创建的github页面地址即可访问
如何添加博客就去看知乎教程吧，我把链接放最后，这篇文章也是我这博客的来源
https://www.zhihu.com/question/20962496/answers/updated
